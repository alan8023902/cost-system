# 快速开始指南

## 一键启动全部服务

### Windows环境

**方式一：一键启动（推荐）**
```bash
双击 quick-start-all.bat
```
将自动启动后端和前端，无需手动操作。

**方式二：分别启动**
```bash
# 1. 启动后端
双击 restart-backend.bat

# 2. 启动前端
双击 start-frontend-react.bat
```

---

## 访问系统

- **前端地址**: http://localhost:38443
- **后端地址**: http://localhost:31943
- **测试账号**: admin / password

## 核心功能

### 1. 项目管理
- 项目列表查看
- 项目详情（版本列表、成员管理）

### 2. 版本工作台
- 高密度表格（行内编辑、批量粘贴）
- 三大模块切换（材料/分包/费用）
- 实时指标计算

### 3. Excel导入
- 拖拽上传Excel文件
- 自动解析数据
- 错误提示

### 4. 指标看板
- 显示关键财务指标
- 实时重算

## 常用操作

### 表格编辑
1. **双击单元格**进入编辑模式
2. **Tab键**切换到下一列
3. **Enter键**切换到下一行
4. **Esc键**取消编辑

### Excel批量粘贴
1. 从Excel复制数据（Ctrl+C）
2. 在表格区域按**Ctrl+V**
3. 自动解析并导入多行数据

### Excel导入
1. 点击"导入Excel"按钮
2. 拖拽或选择Excel文件
3. 点击"开始导入"

## 常见问题

**Q: 登录失败怎么办？**
A: 确认后端已启动，默认账号: admin / password

**Q: Excel导入格式？**
A: 列顺序为：项目名称 | 规格型号 | 单位 | 数量 | 单价 | 税率 | 备注

**Q: 如何查看完整文档？**
A: 查看 `cost-frontend-react/README.md` 或 `docs/` 目录
