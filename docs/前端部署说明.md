# 工程成本计划与税务计控系统 - React前端

## 当前部署说明

### 前端选型
经过评估，已采用**原始React设计**替代Vue前端，原因如下：

1. **UI完全符合需求** - 原始设计即为工程工作台风格（高密度表格、专业克制）
2. **多端支持完整** - 已内置PC/Pad/Mobile响应式适配
3. **实现速度最快** - 无需重新设计UI，直接对接后端API即可
4. **维护成本更低** - 单一前端技术栈，避免Vue/React双维护

### 目录结构
```
cost-system/
├── cost-backend/          # Java后端
├── cost-frontend-react/   # React前端（生产使用）
├── cost-frontend/         # Vue前端（已废弃）
├── 工程成本计划与税务计控系统/ # 原始React设计（模板）
└── docs/                  # 文档
```

### 快速启动

#### 方式一：自动启动（推荐）
双击运行：
```
start-frontend-react.bat
```

#### 方式二：手动启动
```bash
cd cost-frontend-react
npm install
npm run dev
```

访问: **http://localhost:38443**

### 技术栈
- **框架**: React 19 + TypeScript
- **构建**: Vite 6
- **路由**: React Router 7
- **样式**: Tailwind CSS 3
- **图标**: Lucide React
- **Excel**: XLSX

### 多端支持
✅ **PC端**: 完整工作台体验（高密度表格、行内编辑）  
✅ **Pad端**: 表格 + 抽屉编辑模式  
✅ **Mobile端**: 列表 + 详情页模式

### UI特点
- 深色左侧导航（专业工程风格）
- 高密度数据表格（36-40px行高）
- 克制的视觉设计（4px圆角、极弱阴影）
- 工作台布局（Header + LeftNav + Main）
- 状态驱动交互（草稿/审批中/已签发）
