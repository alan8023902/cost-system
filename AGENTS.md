## Skills

### Available local skills
- cost-system-ui-global-standard: Mandatory UI rule set for this repository. Use for any frontend/UI layout, style, interaction, and page-structure task. (file: skills/cost-system-ui-global-standard/SKILL.md)
- cost-system-ui-apple: Legacy visual style guidance. Use only as a secondary reference after applying the global standard. (file: skills/cost-system-ui-apple/SKILL.md)
- cost-system-java: Backend/domain implementation rules for this project. (file: skills/cost-system-java/SKILL.md)

### Default execution rule
- For every frontend-related execution, read `docs/Frontend-UI-Global-Standard.md` first and treat it as the highest-priority UI specification.
- If there is any conflict, `docs/Frontend-UI-Global-Standard.md` takes precedence over other UI guidance.
